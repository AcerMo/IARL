Visualization of Agent Replays

— LIAL: LLM assisted Intention-Aligned Reinforcement Learning

Introduction

This project explores LLM-assisted intention alignment in reinforcement learning. By combining large language models with reinforcement learning, we provide novel frameworks for agent behavior visualization and trajectory analysis.
